# 📱 React Native Navigation Learning

A beginner-friendly React Native project to learn different navigation patterns using **React Navigation** and **Expo Router**.

## 🚀 Topics Covered

* ✅ Stack Navigation
* ✅ Tab Navigation
* ✅ Drawer Navigation
* ✅ Screen Navigation
* ✅ Passing Parameters
* ✅ Navigation Header
* ✅ Navigation Options
* ✅ Nested Navigation
* ✅ Back Navigation

## 🛠️ Tech Stack

* React Native
* Expo
* Expo Router
* TypeScript
* React Navigation

Stack-Navigation/

## 📖 What You'll Learn

* Create multiple screens
* Navigate between screens
* Build Stack Navigation
* Create Bottom Tabs
* Add a Drawer Menu
* Customize headers
* Pass data between screens
* Organize navigation files

## ▶️ Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npx expo start
```
